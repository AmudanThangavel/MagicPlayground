For Loop:

For loops are useful when you want to execute the same code for each item in a given sequence.

With a for loop, you can iterate over any iterable data such as lists, sets, tuples, dictionaries, ranges, and even strings.

Syntax

for val in sequence:
         code in loop

 Here,val is the variable that takes the value of the item inside the sequence on each iteration.

sequence:list, tuple, string or other iterable objects.

FOR LOOPS- sample code:


Example:
chocolates = ["Fabelle","Toblerone","Snickers"]
for i in chocolates:
        print(i)
Output:
Fabelle
Toblerone
Snickers
