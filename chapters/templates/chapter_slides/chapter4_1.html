LOGICAL OPERATORS:

Operators are used to perform operations on values and variables.
These are the special symbols that carry out arithmetic and logical computations.
The value the operator operates on is known as Operand.

Logical NOT Operator:

It returns true if operand is false

Syntax:
not x

Logical NOT Operator

sample code:
    x = True
    c=not x
    print('not x is',c)
Output:
not x is False

Logical OR Operator:

It returns true one of the operands is true

Syntax:
x or y

Logical OR Operator:

sample code:

    x = True
    y = False
    b=x or y
    print('x or y is',b)
Output:
x or y is True

Logical AND Operator:

It returns true if both the operands are true

Syntax:
x and y

Logical AND Operator:

sample code:

  x = True
  y = False
  a=x or y
  print('x and y is',a)
Output:
x and y is False
